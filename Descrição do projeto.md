# 📚 Gerenciador de Biblioteca em Java

Este projeto é uma aplicação simples de **Gerenciador de Biblioteca** desenvolvida em **Java**, com foco em prática de **Programação Orientada a Objetos (POO)**.  
O sistema permite adicionar, remover, listar, emprestar e devolver livros de forma interativa via console.

---

## 🚀 Funcionalidades
- Adicionar novos livros ao catálogo
- Remover livros existentes
- Listar todos os livros cadastrados
- Realizar empréstimos de livros para usuários
- Registrar devoluções de livros
- Menu interativo em loop até o usuário decidir sair

---

## 📂 Estrutura do Projeto
- `AppBiblioteca.java` → Classe principal com o menu e interação com o usuário
- `GerenciadorBiblioteca.java` → Classe responsável por gerenciar os livros (adicionar, remover, listar, emprestar, devolver)
- `Livro.java` → Classe que representa um livro com atributos como título, autor, ano, editora e usuário

---

## 🛠️ Tecnologias Utilizadas
- **Java 17+** (funciona também em versões anteriores)
- **Scanner** para entrada de dados via console
- Estruturas de controle (`switch`, `do/while`, `try/catch`) para fluxo do programa
